# Web Persona Skill

Web Persona 人物角色创建与应用专家 - 基于《赢在用户》全书知识体系构建

## 简介

本仓库是基于 Steve Mulder 和 Ziv Yaar 著《赢在用户：Web 人物角色创建和应用实践指南》(The User Is Always Right) 全书知识体系构建的 Web Persona 人物角色创建与应用专家 Skill。

全书从头到尾仔细阅读后，将核心内容整理为完整的方法论体系，覆盖用户研究规划、数据采集、用户细分、角色创建到商业策略应用的完整闭环。

## 安装

详见 [INSTALL.md](INSTALL.md)

### 快速安装

将 `SKILL.md` 放入你的 Agent 系统 skills 目录即可使用。

## 功能

- **人物角色（Persona）创建与分析** - 从零开始完成人物角色全生命周期
- **基于《赢在用户》方法论** - 遵循经过验证的用户研究最佳实践
- **深度用户行为心理分析** - 挖掘用户需求本质和选择逻辑
- **商业策略应用** - 将人物角色应用于功能定义、信息架构、内容策略

## 使用方式

- **作为 AI Agent 的 Skill**：将 SKILL.md 放入你的 Agent 系统 skills 目录
- **作为知识库参考**：直接阅读 SKILL.md 和 docs/ 下的文档
- **作为团队共享资源**：fork 本仓库供团队使用

## 核心理念

- **用户总是对的**：商业成果取决于用户，必须理解并响应真实用户需求
- **你不是你的用户**：设计者与用户存在目标差异、关注点差异、经验差异
- **通过直接接触了解用户**：不能依赖假设，必须通过研究获得真实洞察
- **研究成果必须可实施**：全面、可共享、易记忆、可付诸行动
